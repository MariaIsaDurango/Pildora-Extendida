# Ciclo de Vida de Modelos ML 🤖

Aplicación interactiva para presentar una clase sobre el ciclo de vida y la serialización de modelos de Machine Learning usando Iris.

## Incluye

- 🔄 Ciclo de vida de un modelo.
- 📦 Serialización de modelos.
- ⚖️ Comparación entre Pickle y Joblib.
- 🎯 Quiz de repaso.
- 🏆 Reto guiado para Google Colab, descargable desde la aplicación.

## Ejecución

```bash
pip install -r requirements.txt
streamlit run app.py
```

Para el reto, abre **Reto Colab**, descarga el archivo `.ipynb` y súbelo a [Google Colab](https://colab.research.google.com/).
